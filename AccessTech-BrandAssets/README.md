This folder contains starter brand assets for Access Tech. Support.
Use this guide to find and apply the logo, QR codes, fonts, and templates.

## Quick steps

### Using the logo on your website
- For dark backgrounds use `logo/access-tech-logo-dark.png`.
- Add `favicon.ico` to your website root and reference it:
  <link rel="icon" href="/favicon.ico" />

### Using the QR codes
- Use `qr-codes/qr-set.png` for printing or digital.
- Keep a quiet margin around the QR when printing.
- Test with your phone camera.

### Fonts
- Download Poppins from Google Fonts:
  https://fonts.google.com/specimen/Poppins
- Add to your site's head:
  <link href="https://fonts.googleapis.com/css2?family=Poppins:wght@400;600;700&display=swap" rel="stylesheet">

### Uploading to GitHub
1. Open your repository on github.com
2. Click Add file → Upload files
3. Drag & drop files & commit to your branch (e.g., final-deploy)
